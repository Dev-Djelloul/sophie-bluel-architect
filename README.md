# Portfolio-architecte-sophie-bluel

Code du projet 6 d'integrateur web.

## Apercu

- Frontend statique du portfolio de Sophie Bluel.
- Backend fourni pour la gestion des projets et l'authentification.

## Lancer le projet

1. Installer les dependances a la racine :
   - `npm install`
2. Lancer le backend :
   - `npm start`
   - (ou depuis `Backend/` via `npm start`)
3. Ouvrir le frontend :
   - `npm run open` (ouvre `index.html`)
   - ou ouvrir `index.html` manuellement dans le navigateur.
